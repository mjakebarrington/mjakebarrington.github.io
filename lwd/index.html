JavaScript

HTML defines the content of a webpage. CSS defines the presentation of a site. Finally, JavaScrit defines the functionality of our pages. It is important to learn JavaScript so we can build responsive, dynamic, and mobile friendly websites.

Lexical Structure

  • JavaScript programs are written using the unicode character set.
  • JavaScript is case-sensitive.
    • let name = "Jake";
let Name = "Mike";
let NAME = "I actually have 2 names...";
// All three of the above variables are entirely different.
  • JavaScript ignores spaces that appear between tokens.
    • let myName="Jake";
// While the above code is technically valid...
let myName = "Jake";
// Adding spaces makes code more readable
